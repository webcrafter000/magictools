# 🚀 Magic Tools Generator

A no-code platform that generates internal tools based on natural language descriptions. Turn your ideas into working tools with just a simple description.

## ✨ Features

- **AI-powered tool generation**
  - Natural language processing
  - Schema generation
  - Validation rules
  - Dashboard configuration

- **📦 Supabase integration**
  - Table creation
  - Data validation
  - Backup functionality

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript
  - React 18
  - TypeScript 5
  - Shadcn UI
  - Tailwind CSS
  - React Query

- **Database**: Supabase
  - PostgreSQL 
  - Edge Functions

- **AI**: Gemini AI
  - Natural language processing
  - Schema generation
  - Data Summarization

- **UI**: Shadcn UI + Tailwind CSS
  - Custom components
  - Responsive design
  - Dark/Light mode
  - Accessibility support

## 🚀 Getting Started

1. Install dependencies:

```bash
npm install

npm run dev

```
This will launch the project, and you can open it in your browser at http://localhost:3000
